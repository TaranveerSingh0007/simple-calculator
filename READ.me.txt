# Simple 2-Digit Calculator in C

Welcome to my repository! This is my very first programming project built using the C language. It is a lightweight, console-based application designed to handle fundamental arithmetic operations between two numbers.

Building this project helped me transition from learning programming theory to writing functional, real-world code.

---

## 🚀 Features & Operations

The calculator takes two numbers as input from the user, prompts for an operator, and accurately computes the result. It fully supports:

*   **Addition (`+`)** - Sums two values.
*   **Subtraction (`-`)** - Calculates the difference between the first and second value.
*   **Multiplication (`*`)** - Computes the product of two values.
*   **Division (`/`)** - Divides the first value by the second value.
